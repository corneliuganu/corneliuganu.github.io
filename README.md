# Site-ul oficial Cornel Iuganu

Acesta este site-ul oficial al fotografului de evenimente **Cornel Iuganu**, specializat în nunți, botezuri, portrete și evenimente corporate.

Site-ul prezintă o selecție de galerii foto, informații despre fotograf și un formular de contact pentru programări și oferte.

Proiectul este găzduit pe GitHub Pages la adresa: `https://corneliuganu.github.io`.
